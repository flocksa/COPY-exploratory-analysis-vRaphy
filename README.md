how to run: 
1. Create code space (green button on github)
2. Install python (bottom right pop up)
3. Run api (nhanes_api.py)
4. Make sure to add the port + make public
5. Run the front end (FrontEndUI)
   a. cd frontend
   b. npm install react-select react-plotly.js plotly.js
   c. npm start
